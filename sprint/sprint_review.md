#Sprint Review

During this sprint, we implemented serveral improvements and core features for our task management application. 

## Features Implemented
- Improved task display format for better readibility (#28)
- Added functionality to connect deadlines to tasks (#27)
- Implemented delete task using index (#24)
- Improved mark task as completed using index (#25)
- Added validation for deadline input (#36)
- Improved overall input validation (#30)
- Implemented persistence of completed task status using JSON (#29)

## Application Capabilities
The application now allows users to do the following: 
- Add, delete, and complete tasks
- View tasks in a clear and structured format
- Set and validate deadlines
- Handle invalid user input
- Save and load task data using JSON
